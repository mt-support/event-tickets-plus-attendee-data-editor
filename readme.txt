Extends Event Tickets Plus, making it possible to edit submitted custom attendee data from within the attendee screen.

== Description ==

Event Tickets Plus makes it possible to collect custom attendee data from ticket buyers. For example, t-shirt sizes or
food allergy information can be collected. This extension makes it possible for site administrators to amend this data after it has been submitted.
